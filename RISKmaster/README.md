# Risk
Risk game in python

Launch GUI.py

Shortcut:
<ul>
<li>ESC : quit</li>
<li>n : next phase</li>
<li>p : next player turn</li>
<li>h : show/hide help menu</li>
<li>c : show/hide continents</li>
<li>u : use cards</li>
<li>d : show/hide quests & cards</li>
<li>rigth click : unselect</li>
<li>mouse wheel : increase/decrease number of units select</li>
</ul>
